SSQ Matcher EXEC 
----------------

1. To run the SSQ Matcher 
	java -jar ssqmatcher.jar <SSQ id>

	e.g. java -jar ssqmatcher.jar "SSQ-1"

2. This will give results like this 

	{'array':[SSQ-476, SSQ-473], 'id':SSQ-1}

	[SSQ-476, SSQ-473] - represents the SSQ ids in the order of relevance. 
	
	SSQ-1 - the given SSQ id
